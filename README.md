# FDF
FdF is a 3D Wireframe Viewer written on plane C. (macOS is only supported)

## Usage

 + `make`
 + `./fdf test_maps/<map>`


| keys | Type |
|:------| :--------|
| ← | moves left |
| → | moves right |
| ↑ | moves up |
| ↓ | moves down |
| h | make map higer |
| l | make map lower |
| / | lower scale|
| ] | increse scale |
| mouse | move with a mouse |
